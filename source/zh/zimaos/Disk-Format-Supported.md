## 引言
随着用户对存储设备需求的多样化，ZimaOS不断优化和扩展其文件系统的支持。本文档旨在为用户提供当前版本（v1.5.1）中磁盘格式支持的全面概述，帮助用户更高效地管理和利用存储资源。

<div class="center">
  
| 文件系统格式      | USB磁盘自动挂载 | 内部磁盘 |
|-------------------|-----------------|---------|
| APFS              | 不支持          | 不支持  |
| HFS/HFS+          | 读/写           | 读/写   |
| ExFAT             | 读/写           | 读/写   |
| FAT16             | 读/写           | 读/写   |
| FAT32             | 读/写           | 读/写   |
| NTFS              | 读/写           | 读/写   |
| EXT4              | 读/写           | 读/写   |
| EXT3              | 读/写           | 读/写   |
| EXT2              | 读/写           | 读/写   |
| BTRFS             | 读/写           | 读/写   |
| XFS               | 不支持          | 不支持  |

</div>

**<u>当前版本 (v1.5.1)</u>**

{% note warn 提示: %}

USB驱动器：可以直接用于保存数据。
内部硬盘：插入Cube后，点击“启用”按钮即可保存数据以供使用。

{% endnote %}
