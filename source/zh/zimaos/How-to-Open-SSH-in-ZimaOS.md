根据社区反馈，我们已简化了在 ZimaOS 中启用 SSH 的过程。为了让这一功能更加方便易用，我们将其直接集成到了 UI 中。现在，访问 SSH 和使用终端变得更加快捷和简单。

**在 ZimaOS v1.2.5 中，启用 SSH 变得比以往任何时候都更简单。按照以下步骤即可访问 SSH：**

### 1. 打开设置。
![](https://manage.icewhale.io/api/static/docs/1769066381958_image.png)

### 2. 导航到开发者模式。

![](https://manage.icewhale.io/api/static/docs/1769066436210_image.png)

### 3. 将 SSH 访问开关切换为“开启”。

![](https://manage.icewhale.io/api/static/docs/1769066449088_image.png)

完成上述步骤后，您可以直接使用 SSH 访问 ZimaOS 系统，无需其他配置。

![](https://manage.icewhale.io/api/static/docs/1769066461468_image.png)

此外，您还可以通过一个开关启用基于 Web 的终端。

![](https://manage.icewhale.io/api/static/docs/1769066753516_image.png)
![](https://manage.icewhale.io/api/static/docs/1769066757583_image.png)

这个简单的更新为需要 SSH 访问的开发者和高级用户提供了更加直接的方式。
