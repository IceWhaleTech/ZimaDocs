## Desbloquear o dispositivo
1. **Remover o painel frontal**
Comece por remover cuidadosamente o painel frontal do chassis do ZimaCube para aceder aos componentes internos.
{% note warn Tips %}
O painel frontal está fixado com ímanes. Para o remover, basta pressionar os cantos para o levantar. 
{% endnote %}

3. **pressionar o botão para desbloquear a pega**
Localize e pressione o botão amarelo situado no compartimento da unidade de disco rígido. Isto irá libertar o mecanismo de bloqueio da bandeja do disco rígido.
4. **retirar a bandeja**
Assim que a pega da bandeja do disco rígido sair, puxe-a suavemente para deslizar lentamente a bandeja para fora do compartimento.
##  Instalar disco SATA
### Instalar disco rígido de 3,5
1. **Preparar a bandeja do disco rígido:**
- Remova a bandeja do disco rígido e prepare-a.
2. **Inserir o disco rígido de 3,5 polegadas:**
- Coloque o disco rígido de 3,5 polegadas na bandeja, alinhando os orifícios dos parafusos nas laterais do disco rígido com os da bandeja.
3. **Fixar o disco rígido:**
- Utilize os parafusos 6-32 fornecidos para fixar o disco rígido no lugar. Recomenda-se utilizar os parafusos fornecidos com o seu ZimaCube para um encaixe adequado.
- Certifique-se de que os parafusos estão apertados para que a superfície fique nivelada e uniforme.
### Instalar disco rígido de 2,5
1. **Preparar a bandeja do disco rígido:**
- Remova a bandeja do disco rígido e prepare-a.
2. **Inserir o disco rígido de 2,5 polegadas:**
- Coloque o disco rígido de 2,5 polegadas na bandeja, alinhando os orifícios dos parafusos nas laterais do disco rígido com os da bandeja.
3. **Fixar o disco rígido:**
- Utilize os parafusos M3 fornecidos para fixar o disco rígido no lugar. Recomenda-se utilizar os parafusos fornecidos com o seu ZimaCube para um encaixe adequado.
- Certifique-se de que os parafusos estão apertados para que a superfície fique nivelada e uniforme.
  
##  Instalar disco M.2 NVMe 
Como instalar uma unidade M.2 NVMe/PCIe
1. **Preparar o 7.º compartimento:**
Remova o 7.º compartimento e prepare-o.
2. **Inserir o SSD PCIe:**
- Alinhe a interface M.2 e deslize cuidadosamente o SSD NVMe para a ranhura num ângulo de 30 a 45 graus, depois pressione-o para baixo até ficar plano. 
（Saberá que está corretamente instalado quando encaixar no lugar com um clique. Não force; se não entrar facilmente, certifique-se de verificar novamente e ajustar o alinhamento em conformidade.）
3. **Fixar o SSD M.2 NVMe/PCIe:**
- Utilize o parafuso adequado para fixar o SSD no lugar.
- 
{% note warn Tips %}
**M key** usually support PCIe x2/x4.
{% endnote %}
