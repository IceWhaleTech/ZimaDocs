Esta versão foca na estabilidade e confiabilidade. Corrigimos problemas existentes e melhoramos os fluxos de trabalho principais relacionados ao armazenamento, RAID, download de imagens Docker e backups.

![](https://manage.icewhale.io/api/static/docs/1768795532211_image.png)


## Corrigido:
Corrigido erro de obtenção de informações para ZimaOS-HD nos Arquivos  
Corrigido falha de login para sub-usuários com senhas contendo caracteres especiais  
Corrigido exibição de armazenamento utilizado como 0 quando o nome do RAID contém espaços  
Corrigido links de download inválidos para imagens Arch/Debian no ZVM  
Corrigido falha ao exibir discos ao criar RAID em certos dispositivos  
Corrigido falha ao migrar para discos formatados em NTFS  
Corrigido erro na atualização de imagens após modificação da versão da imagem  
Corrigido erro de digitação no DLNA Friendly Name

## Otimizado:
Adicionada a opção de selecionar o diretório raiz nos backups  
Otimizada a estratégia de pull de imagens Docker  
Melhorada a exibição de Tags muito longas nas configurações do app  
Otimizada a interface para recuperar o tamanho do backup  
Aprimoradas as mensagens de erro ao instalar imagens privadas ou inexistentes  
Simplificado o processo de desmarcar itens na barra lateral de Arquivos  
Otimizado o fluxo de desmarcação na Lixeira de Arquivos  
Melhorada a velocidade de download de novas versões  
Aprimorada a exibição de nomes de fontes de backup muito longos
